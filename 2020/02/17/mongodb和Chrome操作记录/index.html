chrom 浏览器启动

efaults read -g com.apple.mouse.scaling

你会看到输出的是 “3”，这是初始速度

输入以下代码改变鼠标移动速度

defaults write -g com.apple.mouse.scaling 7

后面的数字 “7” 代表速度因子，建议设置在6-8之间，不然怕你受不住，调完之后重启电脑

hbuildX生成的本地资源目录

/Users/masia/work/project/yzgs_app/yzgs_app/unpackage/resources/UNIA011DA1/www

as本地打包uniapp

  1. 配置应用 主体
    替换 src/main/assets/apps/ 中的 Hello 文件夹 （替换为上面的 UNIA011DA1 文件夹）

  2. 配置应用信息
    修改 src/main/assets/data/ 下面的 dcloud_control.xml 文件中的内容

  3. 配置应用的包名和版本号
    打开 src/main/AndroidManifest.xml 文件

  • package
    其中package为应用的包名，采用反向域名格式，为应用的标识，例如： io.uniapp.yzgs
  • versionCode (没做也成功)
    versionCode为应用的版本号（整数值），用于各应用市场的升级判断，建议与manifest.json中version -> code值一致；(如100)
  • versionName (没做也成功)
    versionName为应用的版本名称（字符串），在系统应用管理程序中显示的版本号，建议与manifest.json中version -> name值一致。（如 1.0.0)
  1. 配置应用名称 （没做也成功）

打开src -> main ->res -> values -> strings.xml文件，修改“app_name”字段值，该值为安装到手机上桌面显示的应用名称，建议与manifest.json中name（基础配置中的应用名称）对应。

  1. 配置应用图标和启动界面 （没做也成功）
    将应用的图标（文件名为icon.png）启动图片按照对应的尺寸拷贝到工程的app->src->main->res -> drawable-XXX目录下。
  • icon 是 应用logo图标 （144 * 144）
  • splash 是 应用的启动图（144 * 144）
  • push 是 应用的推送图标（1080 * 1882）
  1. (非官方, 没做也成功)打开 build.gradle(app), 修改
    android {
    defaultConfig {
    applicationId "io.uniapp.yzgs" // 与 包名 一致
versionCode 1 // 与 mainifest.js 中的 version.code 一致
versionName "1.0" // 与 mainifest.js 中的 version.name 一致
    }
    }
